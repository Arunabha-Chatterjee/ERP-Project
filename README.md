#### **Web-Based ERP System for Centralized Business Operations**



The ERP Management System is a web-based application developed to streamline and manage business operations through a centralized platform. It provides an interactive dashboard for viewing business data over different time periods and monitoring recent activities. The system enables efficient management of business records by allowing users to view and manage product, customer, and invoice information along with their related details and summaries. It also supports maintaining and updating customer records and tracking transactions associated with products and invoices.	 

&#x20;                                    



##### **Features**



###### **Backend**

* **RESTful API Endpoints:** Structured API design for seamless communication between frontend and backend.
* **CRUD Operations:** Robust management of products, customers, and invoices with Create, Read, Update, and Delete functionalities.
* **Data Persistence:** Efficient handling of business data using MongoDB for high scalability and flexibility.
* **Transaction Tracking:** Accurate recording and retrieval of invoice and product-related transactions.
* **Centralized Business Logic:** Consolidation of core operations, validations, and summaries for maintainable and secure backend workflows.



###### **Frontend**

* **Interactive Dashboard:** Dynamic data visualization for business metrics and activity monitoring.
* **Responsive UI Design:** Modern interface leveraging Tailwind CSS for consistent styling.
* **Form Management:** Efficient input validation and record management for products, customers, and invoices.
* **Real-Time Data Rendering:** Immediate reflection of backend changes on the frontend interface.
* **User-Friendly Navigation:** Intuitive menu structure and components for seamless operation across the application.





##### **Tools and Technologies**

###### 

###### **Backend**



**Spring Boot:** A Java-based framework for creating robust and scalable RESTful APIs with minimal configuration. Provides built-in support for dependency injection, security, and database integration.



###### **Frontend**



**React:** A modern JavaScript library for building dynamic and responsive user interfaces. Enables reusable components and efficient state management.



**Tailwind CSS:** Utility-first CSS framework for fast, responsive, and consistent styling of the web application.



###### **Database**

**MongoDB:** A NoSQL document database used for flexible and high-performance data storage, ideal for handling complex ERP data structures.



###### **Development Tools**



**IntelliJ IDEA:** Integrated Development Environment (IDE) used for backend development with Spring Boot, offering intelligent code completion and debugging.



**VS Code:** Lightweight code editor used for frontend development with React, providing extensions for productivity and styling support.



**MongoDB Compass:** GUI tool for managing and visualizing MongoDB collections and performing queries efficiently.



**Postman:** API testing tool to verify RESTful endpoints, test requests, and validate responses.
